* Ruby version
    ```
    rails 3.1.3
    ```

* System dependencies
  - RSpec
  - Shoulda-matcher
  - Pagy
  - devise
  - tailwindcss
 

* Configuration

     ✔️ Clone repository
     ```
     $ git clone @git:github.com/chrystalme/imdbc.git
     ```
     ✔️ Install the dependencies
     ``` 
     $ bundle install
     ```
 * Database creation

      ✔️ Migrate database
    ```
    $ rails db:migrate
    ```

   ✔️ Seed the database
     ```
     $ rails db:seed
     ```
     
* How to run the application

    ```
    $ bin/dev
    ```
    
   ✔️ on your local machine visit 
   
   ```
   http://localhost:3000
   ```
* How to run the test suite

    ```
    $ rspec spec
    ```
